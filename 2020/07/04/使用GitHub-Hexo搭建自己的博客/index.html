绑定域名

首先，你需要有一个域名。如果没有的话可以在阿里云、腾讯云、godaddy、namesilo等网站购买，第一年一般比较便宜，如果想长期使用的话要看清楚续费价格哦。在有.com的域名并且价格可以接受时建议优先选择这个，如果没有的话就看个人喜好，.xyz或者.icu或者.me都是目前比较受欢迎且便宜的域名。在阿里云与腾讯云买的域名需要备案，我比较怕麻烦就在namesilo上购买了。

购买成功后进入域名管理界面，为域名添加解析。

首先，在Cmd中输入ping命令以获得你的仓库的ip地址。

为你的域名添加一条A类型的解析记录，如下图所示。HOSTNAME填写@（但在namesilo填写@会报错，所以不填），IPV4 ADDRESS填写刚刚得到的ip。

再添加一条CNAME类型的解析记录，如下图所示。HOSTNAME填写www，TARGET HOSTNAME填写你的仓库的名字。

完成后保存，登录你的GitHub账号，进入你建立的仓库，点击settings然后一直向下滑动倒GitHub Pages，然后将custom domain改为你的域名并保存。

然后进入本地blog文件夹，在source子文件夹中创建一个文件，输入你的域名并保存（最好不带www）。将文件重命名为CNAME，无后缀。
