HDU2859-Phalanx最大对称子矩阵
Created 2020-07-13| Updated 2020-07-13|题解
Word count:800|Reading time: 3 min
|Post View:

Problem Description

Today is army day, but the servicemen are busy with the phalanx for the celebration of the 60th anniversary of the PRC.
A phalanx is a matrix of size nn, each element is a character (az or AZ), standing for the military branch of the servicemen on that position.
For some special requirement it has to find out the size of the max symmetrical sub-array. And with no doubt, the Central Military Committee gave this task to ALPCs.
A symmetrical matrix is such a matrix that it is symmetrical by the “left-down to right-up” line. The element on the corresponding place should be the same. For example, here is a 33 symmetrical matrix:
cbx
cpb
zcc

Input

There are several test cases in the input file. Each case starts with an integer n (0<n<=1000), followed by n lines which has n character. There won’t be any blank spaces between characters or the end of line. The input file is ended with a 0.

Output

Each test case output one line, the size of the maximum symmetrical sub- matrix.

Sample Output

题意

给出由az或AZ构成的字符矩阵，求它关于反对角线对称的最大子矩阵的边长。

思路

因为对称轴为右上到左下，所以以$dp[i][j]$表示第i行第j列的点为左下顶点时最大对称矩阵的边长，判断每个位置的$dp[i][j]$时依次检查它的上方和右侧的字符是否一一相同，直到与$(i,j)$的点相距$dp[i-1][j+1]$的位置（此时两个位置分别为$dp[i-1][j+1]$代表的最大子矩阵的左上顶点的左侧点和右下顶点的下方点）。

如果该范围内所有字符全部相同，则$dp[i][j]=dp[i-1][j+1]+1$。

如果在距离$(i,j)$点k个长度的位置字符不同，则$dp[i][j]=k$。

在dp数组中取最大值即为所求。

如果要求最大对称矩阵，则找到最大边长的位置输出右上方的最大对称阵即可。

易错点：

  1. 在代码中我以$(i,n-1-j)$来代表思路中位于位置$(i,j)$的点，写代码的时候比较容易出错，建议读者（如果有的话）换成较为直观的$(i,j)$。

  2. 上边和右边上的所有点都应该初始化为1。

  3. ans需要初始化为1

  4. 取最大值要放在if外面

  5. 3和4两条中做到一条就行
